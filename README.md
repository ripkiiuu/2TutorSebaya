# 2TutorSebaya
1. Sistem Tarik Saldo (Withdrawal) Mentor 💰                                                                                                                                                                    
  Di dashboard mentor sudah ada tulisan "Total Saldo Bisa Ditarik". Namun, belum ada tombol atau halaman bagi mentor untuk menarik uang tersebut ke rekening bank/Gopay mereka. Admin juga butuh halaman untuk    
  memproses penarikan ini.                                                                                                                                                                                        
  2. Validasi Pembayaran (Opsional tapi Penting) 💳                                                                                                                                                               
  Saat mahasiswa booking dan upload bukti bayar, saat ini bukti tersebut langsung masuk ke mentor. Untuk platform profesional, idealnya Admin yang melakukan validasi apakah transferannya asli/masuk, baru       
  diteruskan ke mentor (sebagai penengah).                                                                                                                                                                        
  3. Tombol Chat WhatsApp Otomatis 💬                                                                                                                                                                             
  Karena kita sudah menyimpan data Nomor WhatsApp ( no_wa ) di profil pengguna, kita bisa mengubahnya menjadi tombol hijau "Chat via WhatsApp" (menggunakan tautan  wa.me/... ). Ini akan sangat memudahkan       
  mahasiswa janjian dengan mentor tanpa harus save nomor manual.                                                                                                                                                  
  4. Fungsi Export PDF (Laporan Admin) 📄                                                                                                                                                                         
  Di dashboard Admin, kita sudah menaruh tombol "Export PDF". Besok kita bisa membuat script PHP sederhana untuk mencetak laporan transaksi atau daftar mentor, yang biasanya sangat disukai oleh dosen penilai   
  karena terlihat nyata.
