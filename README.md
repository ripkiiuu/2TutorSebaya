# 📚 TutorSebaya

TutorSebaya adalah website marketplace layanan bimbingan belajar yang menghubungkan mahasiswa dengan mentor dalam satu platform. Website ini dirancang untuk mempermudah mahasiswa mencari mentor sesuai bidang keahlian, melakukan pemesanan sesi bimbingan, serta berkomunikasi secara langsung melalui fitur chat.

Website ini dikembangkan sebagai tugas besar mata kuliah **Pemrograman Web** menggunakan **PHP Native** dan **MySQL**.

---

## ✨ Fitur Utama

### 👨‍🎓 Mahasiswa
- Registrasi dan Login akun
- Mencari mentor berdasarkan mata kuliah
- Melihat profil dan detail mentor
- Booking jadwal bimbingan
- Upload bukti pembayaran
- Melihat jadwal bimbingan
- Melihat riwayat booking
- Chat dengan mentor
- Memberikan rating dan ulasan
- Mengelola profil
- Menerima notifikasi

---

### 👨‍🏫 Mentor
- Registrasi sebagai mentor
- Verifikasi data mentor
- Melihat permintaan booking
- Menerima atau menolak booking
- Mengelola jadwal bimbingan
- Menambahkan link meeting
- Memberikan catatan hasil bimbingan
- Chat dengan mahasiswa
- Melihat daftar siswa
- Melihat penghasilan
- Mengajukan penarikan saldo
- Mengelola profil

---

### 👨‍💼 Admin
- Dashboard Admin
- Kelola pengguna
- Verifikasi mentor
- Kelola transaksi
- Kelola pengajuan penarikan
- Monitoring aktivitas sistem

## 🚀 Cara Menjalankan

1. Clone repository

```bash
git clone https://github.com/username/TutorSebaya.git
```

2. Pindahkan project ke folder **htdocs**.

3. Jalankan **Apache** dan **MySQL** melalui XAMPP.

4. Import database

```
tutorsebaya.sql
```

5. Buka browser

```
http://localhost/TutorSebaya
```
